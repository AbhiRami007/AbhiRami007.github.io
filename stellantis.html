Free2Move Charge

EV Charging Platform — Stellantis (Maserati)

📌 Overview

Free2Move Charge is an EV charging platform that allows users to:

  • Find nearby charging stations
  • Start and stop charging
  • Pay for charging
  • View charging history and payments

🧠 What the System Does

The platform mainly handles:

  1. Finding charging stations
  2. Managing charging sessions
  3. Handling payments and history

🔄 Overall System Flow

🔋 Charging & Payment Flow

💡 Complete User Flow

1. User Registration / Login
  • User creates an account or logs in
  • Profile and details are saved
  • User enters dashboard
2. Find Charging Stations
  • Nearby stations or clusters are shown
  • User views details
  • User selects a station
3. Start Charging
  • User starts charging
  • Session is created
  • Charging begins
4. Charging in Progress
  • System tracks time and energy usage
  • User monitors progress
5. Stop Charging
  • User stops or auto ends
  • Session closes
  • Usage calculated
6. Payment
  • Cost calculated
  • Payment processed
  • Status updated
7. Save Record
  • Session and payment stored
  • Usage history saved
8. Dashboard / History
  • View history
  • Payment details
  • Usage insights
